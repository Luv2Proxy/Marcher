# Marcher

Babylon.js marching-cubes-style terrain explorer with real-time terrain mining/building inspired by Astroneer.

## Run

```bash
python3 -m http.server 4173
```

Then open http://localhost:4173.

## Controls

- `WASD` + mouse: move and look
- `Shift`: sprint
- `Left click`: mine/retract terrain
- `Right click`: build terrain
- `Q/E`: brush radius down/up
- `Z/X`: brush strength down/up
- `R`: regenerate world
